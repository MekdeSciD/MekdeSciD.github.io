Natural Language Processing

This independent project is developed with the larger application of Natural Language Processing (NLP) in mind. Specifically, in tasks that require text parsing.
Using machine learning to identify similar context within a corpus has an impactful application.
Therefore, this project aims to find the optimal machine learning method to identify a topic with similar content from a given corpus; using one of the most widely translated texts in the world, The Bible.

Economic Impact of a Disaster

This group project looks at ways to measure the economic impact of a disaster in terms of wages and unemployment.
In order to approach this problem, we took a look at an isolated natural disaster and investigated the economic impact that the disaster had on different localities. Hurricane Sandy was used as a case study.
